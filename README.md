# Pokemon Card Browser

A local browser for the Pokemon Trading Card Game collection. Browse 18,000+ cards across 146 sets with rarity data from Scrydex.

## Features

- Browse all Pokemon TCG cards in a single interface
- 18,000+ cards from 146 different sets
- Rarity information from Scrydex
- Local image viewer with downloaded card images
- Fast, responsive card search and filtering

## Project Structure

- `pokemon_card_browser.html` — Main web interface for browsing cards
- `launch_pokemon_browser.sh` — Shell script to launch the card browser
- `scrape_scrydex_rarities.py` — Script to fetch rarity data from Scrydex
- `download_pokemon_images.py` — Script to download card images locally
- `cards.json` — Card database (generated by scripts)
- `cards_by_set/` — Folder containing card images organized by set

## Getting Started

1. Run the Python scripts to generate the card database and download images:
   ```bash
   python3 scrape_scrydex_rarities.py
   python3 download_pokemon_images.py
   ```

2. Launch the browser:
   ```bash
   ./launch_pokemon_browser.sh
   ```

3. Open the provided localhost URL in your browser to start browsing.

## Requirements

- Python 3.6+
- Requests library (for Python scripts)
- Modern web browser
