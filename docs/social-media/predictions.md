## How to Add a Prediction

1. Fill in a new row in the **Active Predictions** table above
2. Run `reminder-script.applescript` (or copy the template and run via Script Editor / Terminal) to create the Apple Reminder
3. When the reminder fires, move the row to **Followed Up** and update Status

### Stance Values
- **Buy** — Strong conviction, buy now
- **Hold** — Already in, staying put
- **Watch** — Monitoring, not buying yet
- **Avoid** — Pass on this one

### Status Values
- **Pending** — Awaiting follow-up date
- **Followed Up** — Post published, outcome documented
