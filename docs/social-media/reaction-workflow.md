# Future Sight — Reaction Workflow (OpenClaw)

How to use OpenClaw to draft a reaction post to someone else's content.

---

## When to Use This

You've seen a video, tweet, or post from another creator about a Pokémon TCG product and want to:
- React publicly with your own take
- Validate or push back on their claim with data
- Tag them and drive conversation

---

## Step-by-Step

### Step 1 — Send the content to OpenClaw

Say something like:

> "Here's a video I want to react to: [URL]"
> "I want to react to this tweet: [URL or paste the text]"
> "Someone just posted that [claim]. I want to respond."

If you don't have a URL (e.g. you saw a video on your feed), describe it:
> "Someone on Threads said Prismatic Evolutions ETBs are a bad investment right now. I want to push back."

---

### Step 2 — OpenClaw asks what you need

OpenClaw will respond:

> "What are you looking for?
> 1. Draft a reaction post
> 2. Research the claim or product
> 3. Find the creator's handle to tag
> 4. All of the above"

Pick one or say "all." You can also say "just give me the research first."

---

### Step 3 — OpenClaw researches and drafts

OpenClaw will:
- Query the Pokemon Card Browser for card/set data if a product is mentioned
- Pull any relevant price context from Future Sight data
- Summarize what's accurate, inaccurate, or worth nuancing in the original claim
- Draft a ready-to-edit post following the Future Sight voice guide
- Include the creator's handle (if found) for tagging

---

### Step 4 — You review, edit, post

The draft is a starting point. Apply your own voice, trim if needed for platform limits, and post.

After posting: log in `post-archive.md` (Type: Reaction).

---

## Tips

- If a product name is mentioned, spell it out clearly — OpenClaw needs to match it to database records
- If you want to push back hard, say so: "I want to disagree with this take"
- If you want to be diplomatic, say so: "I want to add nuance, not attack them"
- You can ask OpenClaw to produce a Threads version and an X version if character limits matter

---

## Example Prompt to OpenClaw

> "Here's a tweet: 'Surging Sparks boxes are already peaking, sell now if you have them.' I want to react. Find the creator's handle and draft a post that pushes back — I think there's more runway. Use Future Sight data if you have it."
